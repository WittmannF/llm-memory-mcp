# Memory Strategy Benchmark

Benchmark/hackathon framework for comparing memory strategies in this monorepo.

## Goal

Evaluate which strategy best serves as memory for language models under realistic constraints:

- quality;
- faithfulness;
- cost;
- latency;
- update behavior;
- context extraction quality;
- human inspectability.
